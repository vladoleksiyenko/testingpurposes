# Product API
This is a REST API for managing products in a database. It allows users to retrieve, add, update, and delete products.

# Features
- Get all products
- Get a product by ID
- Get products by category
- Add a new product
- Update an existing product
- Delete a product

# API Endpoints

Show all products
- GET /api/products

Show product by id
- GET /api/products/:id

Show products in category
- GET /api/products/category/:category

Send new product
- POST /api/products

Update product
- PUT /api/products/:id

Delete product
- DELETE /api/products/:id
